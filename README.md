# Insideout Projects Hackathon Demo

Frontend prototype สำหรับเว็บไซต์ร้านอาหาร/คาเฟ่ พัฒนาด้วย Next.js และ TypeScript โดยเน้น UI ที่ดู modern, ใช้งานง่าย, และ responsive

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Run Project

```bash
npm install
npm run dev
```

เปิดใช้งานที่ `http://localhost:3000`
